*** some Array Function In Js ***
arr.some(callback function)
Method Javascript
 *return Boolean : true-false 
 *Test at least one element in the array pass the test(condition) in the callback function
 *It doesn't modify the array
 --Example--
 let myArr = [1,2,3,4,5];
 let isEven = myArr.some((elem)=> elem % 2 === 0);
 console.log(isEven); //true
